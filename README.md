# 🚀 Java Web 全栈开发学习路线与实战记录

## 📖 仓库简介

本仓库用于记录完整 Java Web 开发体系的学习历程。涵盖从前端基础（HTML/CSS/JS/Vue）、后端架构（Spring Boot/Maven）、数据库操作（MySQL/MyBatis）到项目部署（Linux）的核心技术栈。仓库内包含了每日的学习笔记、随堂代码以及阶段性的综合实战项目。

## 🛠️ 技术栈总览

* **前端技术**：HTML5, CSS3, JavaScript, Vue3, TypeScript, Element Plus
* **后端技术**：Java, Spring Boot, Spring MVC, Spring Framework (IOC/DI, AOP)
* **构建与管理**：Maven, Git
* **数据库**：MySQL
* **部署环境**：Linux
