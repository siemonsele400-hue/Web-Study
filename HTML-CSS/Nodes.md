# Day 01 - 前端 Web 基础 (HTML & CSS)

## 一、 Web 前端基础与工作原理
* **Web 标准组成**：
  * **HTML**：负责网页的结构（页面元素和内容）。
  * **CSS**：负责网页的表现（页面元素的外观、位置等页面样式，如：颜色、大小等）。
  * **JavaScript**：负责网页的行为（交互效果）。
* **网页的本质**：网页本质上是程序员写的前端代码。这些代码通过浏览器转换成用户看到的网页。
* **浏览器内核**：浏览器对代码进行解析渲染的部分称为浏览器内核。不同浏览器内核不同，对于相同的前端代码解析的效果可能会存在差异。

---

## 二、 HTML 核心概念与基础
* **HTML 概念**：HTML（HyperText Markup Language）即超文本标记语言。它由预定义好的标签构成，如 `<h1>` 展示标题，`<img>` 展示图片等。HTML 代码直接在浏览器中运行，由浏览器解析。
* **HTML 标签特点**：
  * html 标签不区分大小写，但建议小写。
  * 属性值使用单引号或双引号都可以。
  * 语法结构松散，建议规范书写。
* **页面基本骨架**：包含 `<html>` 根标签、`<head>`（网页头部，存放给浏览器看的信息，如 CSS 样式）、`<title>`（网页标题）以及 `<body>`（网页主体，存放给用户看的信息，如文字、图片等）。

## 三、 常见 HTML 标签及其用途

### 1. 排版与文本标签
* **标题标签**：`<h1>` - `<h6>`，用于定义页面标题。
* **段落标签**：`<p>`，用于定义文本段落。
* **文本格式化标签**：
  * **加粗**：`<b>` / `<strong>`（其中 `<strong>` 具有强调语义）。
  * **下划线**：`<u>` / `<ins>`（其中 `<ins>` 具有强调语义）。
  * **倾斜**：`<i>` / `<em>`（其中 `<em>` 具有强调语义）。
  * **删除线**：`<s>` / `<del>`（其中 `<del>` 具有强调语义）。

### 2. 媒体与超链接标签
* **图片标签 (`<img>`)**：属性包括 `src`（指定图片路径）、`width` 和 `height`。
* **视频标签 (`<video>`)**：属性包括 `src`（指定视频的 URL）、`controls`（是否显示播放控件）、`width` 和 `height`。
* **超链接标签 (`<a>`)**：`href` 指定资源访问的 url；`target` 指定在何处打开资源链接（`_self` 为默认值，在当前页面打开；`_blank` 在空白页面打开）。

### 3. 表单标签 (`<form>`)
* **作用**：在网页中主要负责数据采集功能，如注册、登录等。
* **核心属性**：
  * `action`：规定当提交表单时向何处发送表单数据（URL）。
  * `method`：规定用于发送表单数据的方式（GET、POST）。GET 方式表单数据拼接在 url 后面，大小有限制；POST 方式表单数据在请求体中携带，大小没有限制。
* **常用表单项**：
  * `<input>`：定义表单项，通过 `type` 属性控制输入形式。
  * `<button>`：定义按钮（如 submit、reset、button）。
  * `<select>`：定义下拉列表。
  * `<textarea>`：定义文本域。

### 4. 表格标签 (`<table>`)
* `<table>`：定义表格整体。
* `<thead>`：用于定义表格头部（可选）。
* `<tbody>`：定义表格中的主体部分（可选）。
* `<tr>`：表格的行，可以包裹多个 `<td>`。
* `<td>`：表格普通单元格，如果是表头单元格可替换为 `<th>`。

### 5. 无语义布局标签
* **`<div>`**：一行只显示一个（独占一行）；宽度默认是父元素的宽度，高度默认由内容撑开；可以设置宽高（width、height）。
* **`<span>`**：一行可以显示多个；宽度和高度默认由内容撑开；不可以设置宽高。

## 四、 CSS 样式控制与排版

### 1. CSS 引入方式
* **行内样式**：写在标签的 `style` 属性中（配合 JavaScript 使用）。
* **内部样式**：写在 `<style>` 标签中（通常约定写在 `<head>` 标签中）。
* **外部样式**：写在一个单独的 `.css` 文件中，通过 `<link>` 标签在网页中引入。

### 2. CSS 选择器
选择器用来选取需要设置样式的元素。优先级顺序为：**id选择器 -> 类选择器 -> 元素选择器**。
* **元素选择器**：`元素名称 {...}`，如 `h1 {...}`。
* **类选择器**：`.class属性值 {...}`，如 `.cls {...}`。
* **ID 选择器**：`#id属性值 {...}`，如 `#hid {...}`。
* **分组选择器**：`选择器1,选择器2 {...}`，如 `h1,h2 {...}`。
* **属性选择器**：`元素名称[属性名="值"] {...}`，如 `input[type="text"] {...}`。
* **后代选择器**：`元素1 元素2 {...}`，如 `form input {...}`。

### 3. 颜色表示形式
* **关键字**：如 red、green、blue。
* **rgb 表示法**：`rgb(r,g,b)`，红绿蓝三原色，取值 0-255。
* **rgba 表示法**：`rgba(r,g,b,a)`，a 表示透明度，取值 0-1。
* **十六进制表示法**：`#rrggbb`，将数字转换成十六进制表示。

### 4. 盒子模型 (Box Model)
页面中所有的元素都可以看做是一个盒子，由盒子将页面中的元素包含在一个矩形区域内。
* **组成部分**：内容区域（content）、内边距区域（padding）、边框区域（border）、外边距区域（margin）。
* 设置边距属性时，可以按上、右、下、左的顺序设置，也可以通过 `padding-top`、`margin-left` 等单独指定某一个方位。

### 5. Flex 弹性布局
Flex 是一种一维的布局模型，可以为元素之间提供强大的空间分布和对齐能力。
* **开启方式**：`display: flex;`。
* **主轴方向 (`flex-direction`)**：
  * `row`：主轴方向为 x 轴，水平向右（默认）。
  * `column`：主轴方向为 y 轴，垂直向下。
* **主轴对齐方式 (`justify-content`)**：
  * `flex-start`：从头开始排列。
  * `flex-end`：从尾部开始排列。
  * `center`：在主轴居中对齐。
  * `space-around`：平分剩余空间。
  * `space-between`：先两边贴边，再平分剩余空间。
